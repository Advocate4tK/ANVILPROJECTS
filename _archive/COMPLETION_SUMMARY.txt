🗄️ DATABASE DESIGN
═══════════════════════════════════════════════════════════════════════

5 INTERCONNECTED TABLES:

Table 1: REFEREES (13 fields)
  ├─ Personal information (name, email, phone)
  ├─ Certification level
  ├─ Travel preferences
  ├─ Links to: Clubs, Availability, Games
  └─ Status tracking

Table 2: CLUBS (10 fields)
  ├─ Club details
  ├─ Assignor contact information
  ├─ Regional categorization
  ├─ Links to: Referees, Fields, Games
  └─ Notes

Table 3: FIELDS/LOCATIONS (11 fields)
  ├─ Field name and address
  ├─ GPS coordinates
  ├─ Parking and facility information
  ├─ Links to: Club, Games
  └─ Notes

Table 4: GAMES (15 fields)
  ├─ Date, time, location
  ├─ Team information
  ├─ Age group and division
  ├─ Referee assignments (Center, AR1, AR2)
  ├─ Links to: Field, Club, Referees
  ├─ Game status tracking
  └─ Payment tracking

Table 5: AVAILABILITY (16 fields)
  ├─ Referee information from form
  ├─ Date and time range
  ├─ Location preferences (multiple select)
  ├─ Position preferences (multiple select)
  ├─ Age group preferences (multiple select)
  ├─ Links to: Referee record
  ├─ Status tracking
  └─ Automatic timestamp

📋 NEXT STEPS
═══════════════════════════════════════════════════════════════════════

TO GET STARTED:

Step 1: READ
  📄 Open START_HERE.txt (overview)
  📄 Then read QUICK_START.md (30-min setup)

Step 2: SETUP AIRTABLE (15 minutes)
  📄 Follow SETUP_GUIDE.md
  • Create free account at airtable.com
  • Create "Referee Scheduling" base
  • Add 5 tables with all fields
  • Get API token and Base ID

Step 3: CONFIGURE (5 minutes)
  💻 Run create-config.bat (Windows)
  💻 Run ./create-config.sh (Mac/Linux)
  • Creates config.js from template
  • Edit config.js with your credentials

Step 4: TEST (5 minutes)
  🌐 Open index.html in browser
  • Navigate to form
  • Submit test availability
  • Verify in Airtable

Step 5: DEPLOY (5 minutes)
  🚀 Go to netlify.com/drop
  • Drag entire folder
  • Get live URL
  • Share with referees

TOTAL TIME: 30 minutes to full deployment! ⚡
