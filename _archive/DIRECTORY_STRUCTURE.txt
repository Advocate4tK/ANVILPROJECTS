FILE DESCRIPTIONS
=================

index.html
  Main landing page with navigation to form and admin dashboard.
  Open this first to see the system.

referee-availability-form.html
  The form referees use to submit their availability.
  Includes fields for dates, times, locations, positions, etc.

config.template.js
  Template configuration file.
  COPY THIS to config.js and add your Airtable credentials.
  DO NOT edit this template directly.

create-config.bat (Windows)
create-config.sh (Mac/Linux)
  Helper scripts to create config.js from template.
  Double-click (Windows) or run in terminal (Mac/Linux).

css/styles.css
  All styling for the entire application.
  Responsive design, works on mobile and desktop.

js/airtable-client.js
  JavaScript module that communicates with Airtable API.
  Handles creating, reading, updating records.

js/form-handler.js
  JavaScript that handles form submission.
  Validates input, formats data, sends to Airtable.

airtable/base-structure.json
  Complete schema for Airtable database.
  Shows all tables, fields, relationships in JSON format.

airtable/field-definitions.md
  Detailed documentation of every field in every table.
  Use this as reference when setting up Airtable.

README.md
  Main documentation file.
  Overview of system, features, setup instructions.

QUICK_START.md
  Fast-track setup guide.
  Follow this to be up and running in 30 minutes.

SETUP_GUIDE.md
  Detailed step-by-step Airtable configuration.
  Shows exactly how to create tables and fields.

DEPLOYMENT_GUIDE.md
  Instructions for deploying to various hosting platforms.
  Includes Netlify, Vercel, GitHub Pages, etc.

DEPLOYMENT_CHECKLIST.md
  Checklist to verify everything before going live.
  Use this to ensure nothing is missed.

PROJECT_SUMMARY.md
  Complete overview of the entire project.
  Features, architecture, use cases, etc.

EXPORT_INSTRUCTIONS.md
  How to package and transfer this project.
  Instructions for ZIP files, cloud storage, etc.

.gitignore
  Tells Git which files to ignore.
  Protects config.js from being committed.

LICENSE.txt
  MIT License - free to use and modify.


MISSING FILE? CREATE config.js
===============================
If you don't see config.js, that's NORMAL!
You need to create it from config.template.js

Method 1: Use helper script
  Windows: Double-click create-config.bat
  Mac/Linux: Run ./create-config.sh in terminal

Method 2: Manual copy
  Windows: copy config.template.js config.js
  Mac/Linux: cp config.template.js config.js

Then edit config.js and add your Airtable credentials.
